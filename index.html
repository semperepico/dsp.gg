My main source of study material is The Odin Project

The Odin Project is a great project based curriculum which you can use to learn the foundations of HTML, CSS, and JavaScript and then choose to specialise in JavaScript using React and Node or Ruby with Ruby on Rails. I highly recommend it as the curriculum mixes original content with some of the best resources on the internet for learning web development.

But, instead of doing the random generic projects that it suggests, I am applying the instructions to creating my own website from scratch.

Here's what I learned today:

  1. CSS Specificity
    • ID selectors have a higher specificity score than class selectors which have a higher specificity score than element selectors
    • If there is a tie-break in specificity ranking then the last written style (measured from top to bottom) in the CSS will be applied
    • Universal selectors (like '*') have no specificity ranking
  2. How to publish my website using GitHub Pages
  3. Some VS Code shortcuts such as how to change multiple lines at a time (option+shift+i on mac) and opening a terminal window (control+shift+backtick)
